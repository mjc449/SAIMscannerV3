3 // SAIMScannerV3 - an open-source microscope controller providing an //
4 // embedded solution for hardware synchronization. //
5 // This library provides a compiler independent interface //
6 // with the controller hardware on Windows systems. //
7 // Many functions are Scanning Angle Interference Microscopy //
8 // (SAIM) specific, however the hardware and underlying //
9 // functionality is designed to be versatile and applicable //
10 // in a variety of applications. //
11 // //
12 // While devices can be created in applications using a //
13 // variety of methods, the recommended approach is using a //
14 // smart pointer, e.g.: //
15 // //
16 // #include <functional> //
17 // typedef std::shared_ptr<SSV3::Controller> pScanCard; //
18 // pScanCard scancard(SSV3::CreateDevice(), //
19 // std::mem_fn(&(SSV3::Controller::Destroy))); //
20 // //
21 // In this way the controller is shutdown to a safe state //
